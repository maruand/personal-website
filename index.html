Sentiment Analysis of Tweets

Showcasing deep learning and machine learning techniques.

Project Overview

This project analyzes the sentiment of tweets using various machine learning and deep learning techniques. The goal is to classify tweets as positive, negative, neutral or irrelevant.

Techniques Used

  • Preprocessing: Data cleaning, embedding using Google's Universal Sentence Encoder
  • Machine Learning: Logistic Regression, SVM, Random Forest
  • Deep Learning: LSTM, GRU, BERT
